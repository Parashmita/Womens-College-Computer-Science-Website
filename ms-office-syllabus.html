Certificate Course in Microsoft Office

Duration: 6 Months / 72 Hours

Programme Outcome

This course introduces students to computer fundamentals, MS Windows, MS Word, Excel, and PowerPoint. It’s designed for beginners with no prior computer knowledge. Students will gain hands-on skills in office applications essential for career and academic growth.

Detailed Syllabus

Unit Topic Theory (Hours) Practical (Hours)
I Fundamentals of Computer: History, characteristics, components, generations, memory types (RAM, ROM), storage devices, types of software. 2 0
II MS-Windows: Desktop, taskbar, windows accessories, control panel, file/folder management. 3 9
III Documentation using MS-Word: Creating/editing documents, formatting, headers/footers, tables, mail merge, file management, printing. 5 13
IV Electronic Spreadsheet using MS-Excel: Worksheet creation, formulas/functions, data formatting, charts, sorting/filtering. 6 16
V Presentation using MS-PowerPoint: Creating/editing slides, text formatting, inserting objects, transitions, animations, printing handouts. 4 14
